# 📊 Sales Data Analysis Project (Python)

## 🚀 Overview

This project analyzes sales data using Python to extract meaningful business insights.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib

## 📈 Features

* KPI Calculation (Revenue, Orders, Customers)
* Category-wise Sales Analysis
* Monthly Trend Analysis
* Data Visualization (Bar, Line, Pie Charts)
* Business Insights Generation

## 📊 Sample Insights

* Electronics category generates the highest revenue
* Sales peak towards the end of the year
* Average order value calculated for business analysis

## ▶️ How to Run

```bash
pip install pandas matplotlib
python analysis.py
```

## 💡 Project Purpose

To demonstrate data analysis skills and ability to derive actionable insights from raw data.
